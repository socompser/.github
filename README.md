

A **Socompser** é uma empresa angolana especializada em soluções empresariais integradas (ERP), com foco na **Plataforma MASTER**.

Transformando processos empresariais em experiências digitais eficientes e seguras.

---

## Plataforma MASTER

A **Plataforma MASTER** é um sistema ERP modular e escalável, desenvolvido para otimizar a gestão empresarial.  
Ela integra módulos de:

- Contabilidade e finanças  
- Gestão administrativa  
- Gestão de stock e património  
- Processos clínicos  

O sistema permite operações centralizadas, automação de processos e relatórios avançados, garantindo **eficiência, confiabilidade e segurança** para organizações de qualquer tamanho.

---

## Tecnologias e Ferramentas

- Backend moderno com escalabilidade  
- Frontend intuitivo e responsivo  
- Banco de dados seguro e robusto  
- CI/CD e deploy automatizados  
- Gestão de código via GitHub
